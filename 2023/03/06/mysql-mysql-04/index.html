列的数据类型

数值

  • tinyint 非常小的正整数 1个字节
  • smallint 小的整数 2个字节
  • mediumint 中等大小的整数 3个字节
  • int 标准整数 4个字节
  • bigint 大的整数 8个字节
  • float 单精度浮点数 4个字节
  • double 双精度浮点数 8个字节
  • decimal 用于一些经济计算

字符

  • char
  • varchar
  • tinyblob
  • tinytext
  • blob
  • text

日期

  • date
  • time
  • datetime
  • timestamp 时间戳

null

数据库字段属性

扩展

每一个表都要有以下五个字段

  • id 主键
  • version 乐观锁
  • is_delete 未删除
  • gmt_create 创建时间
  • gmt_update 修改时间
