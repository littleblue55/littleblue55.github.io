Java-用户交互scanner

Feb 9, 2023

Scanner对象

  1. 创建一个扫描器对象，用于接收键盘数据
  2. 用next或nextLine方式接收
  3. 关闭扫描器：凡是属于IO流（输入输出流）的类如果不关闭会一直占用资源，要养成好习惯用完关掉。

next 和 nextLine 的区别

next：

  1. 一定要读取到有效字符后才可以结束输入
  2. 对输入有效字符之前遇到的空白，next()方法会自动将其去掉
  3. 只有输入有效字符后才将其后面的空白作为分割符或者结束符
  4. next()不能得到带有空格的字符串

nextLine：

  1. 以enter为结束符，返回的是输入回车键之前的所有字符
  2. 可以获得空白
