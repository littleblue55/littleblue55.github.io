数据类型转换

  • 强制转换 格式： （类型）变量名 高 ==> 低
  • 自动转换 低 ==> 高

注意点：

  • 1.不能对布尔值进行转换
  • 2.不能把对象类型转换成不相干的类型
  • 3.在把高容量转换成低容量时，需要强制转换
  • 4.转换的时候可能会出现内存溢出，或者精度问题
