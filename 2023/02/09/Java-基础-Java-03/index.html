Java-运算符

Posted on 2023-02-09 Edited on 2023-02-10 In Java

Java语言支持一下运算符

  1. 算术运算符：+ - * / % ++ –
  2. 赋值运算符：=
  3. 关系运算符：> < >= <= == != instanceof
  4. 逻辑运算符： && || !
  5. 位运算符：& | ^ ~ >> << >>>
  6. 条件运算符：? :
  7. 扩展赋值运算符：+= -= *= /=
