element-ui的el-dialog的遮罩层在模态框的前面bug

Littleblue
2023-02-08
Vue
Vue, element-ui

原因： 是因为遮罩层插入进了body标签下，z-index高于当前父元素。

解决方法：

在el-dialog标签里加上**:modal-append-to-body=”false”**就可以了。
