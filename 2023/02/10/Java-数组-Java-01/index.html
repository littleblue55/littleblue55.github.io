数组的定义

  • 数组是相同类型数据的有序集合。
  • 数组描述的是相同类型的若千个数据,按照一定的先后次序排列组合而成。
  • 其中，每一个数据称作一个数组元素每个数组元素可以通过一个下标来访问它们。
