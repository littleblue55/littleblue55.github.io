顺序结构

  • Java的基本结构就是顺序结构，除非特别指明，否则就按照顺序一句一句执行。
  • 顺序结构是最简单的算法结构
  • 语句与语句之间，框与框之间是按从上往下的顺序执行的，它是由若干个依次执行的处理步骤组成的，它是任何一个算法都离不开的一种基本算法结构
